Níže si můžete spočítat ověřit finanční zdraví společnosti na základě kalkulace zvoleného bankrotního modelu. Jako vstupní údaje můžete použít veřejně dostupné údaje obchodních společností ze obchodního rejstříku.


Altmanův bankrotní model pro akciové společnosti. Jedná se o klasickou verzi Altmanova bankrotního modelu.
Z = 1,2X1 + 1,4X2 + 3,3X3 + 0,6X4 +1,0X5

Altmanův model pro společnosti s ručením omezeným Odkaz

Zadejte vstupní finanční údaje:

Oběžná aktiva
Krátkodobé zdroje
Celková aktiva
Zisk po zdanění
Zisk před zdaněním a úroky
Tržní hodnota vlastního kapitálu
Celkové tržby

Hodnocení výsledků:
Z > 2,99 → uspokojivá finanční situace firmy
1,81 < Z ≤ 2,99 → šedá zóna neurčitých výsledků
Z ≤ 1,81 → firma je ohrožena bankrotem
