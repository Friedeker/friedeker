Unten können Sie die finanzielle Gesundheit eines Unternehmens anhand der Berechnung des ausgewählten Insolvenzmodells überprüfen. Als Eingabedaten können öffentlich zugängliche Daten von Handelsgesellschaften aus dem Handelsregister verwendet werden.


Altman-Insolvenzmodell für Aktiengesellschaften. Dies ist die klassische Version des Altman-Insolvenzmodells.
Z = 1,2X1 + 1,4X2 + 3,3X3 + 0,6X4 +1,0X5

Altman-Modell für Gesellschaften mit beschränkter Haftung Link

Geben Sie die finanziellen Eingabedaten ein:

Umlaufvermögen
Kurzfristige Verbindlichkeiten
Gesamtvermögen
Gewinn nach Steuern
Gewinn vor Steuern und Zinsen
Marktwert des Eigenkapitals
Gesamteinnahmen

Bewertung der Ergebnisse:
Z > 2,99 → zufriedenstellende finanzielle Lage des Unternehmens
1,81 < Z ≤ 2,99 → Grauzone mit unklaren Ergebnissen
Z ≤ 1,81 → das Unternehmen ist von Insolvenz bedroht
